3AI 자동화 처리에 관한 안내

앱은 Google Gemini AI를 통해 약 봉지 이미지를 자동 분석하여 약품명, 복용 횟수 등을 파싱합니다.

이용자는 AI 자동 처리 결과에 대해 다음 권리를 행사할 수 있습니다.

  • 처리 근거 및 방식에 대한 설명 요청
  • 결과가 부정확한 경우 이의 제기 및 인식 결과 화면에서 직접 수정

4이용자의 권리

현재 약품 복용 정보는 이용자의 기기에만 저장되며, 개발자는 해당 데이터에 접근할 수 없습니다.

권리행사 방법
수정·삭제앱 내 약 관리 화면에서 직접 수정 및 삭제 가능
전체 삭제앱 삭제 시 기기 내 저장 데이터 전체 삭제됨
동의 철회앱 내 설정 화면에서 언제든지 항목별 철회 가능
AI 처리 이의 제기인식 결과 화면에서 직접 수정 가능. 추가 문의는 보호책임자 이메일로 요청
향후 서버 저장 기능이 추가될 경우, 열람·수정·삭제 요청 절차를 처리방침 개정을 통해 별도 안내합니다.

5동의 철회 안내

이용자는 언제든지 앱 내 설정 화면에서 다음과 같이 동의를 철회할 수 있습니다.

동의 항목철회 시 영향
건강정보 처리 동의 기존 저장 데이터는 유지되며 신규 저장이 중단됩니다. 앱의 핵심 기능(복약 알람) 이용이 어려울 수 있습니다.
AI 외부 전송 동의 스캔 기능은 오프라인 OCR 방식으로만 동작합니다.
서비스 분석 동의 앱 기능에 영향이 없습니다.
맞춤형 광고 동의 비개인화 광고로 전환됩니다. 광고 자체는 유지됩니다.

6개인정보의 안전성 확보조치

  • 기기 내 로컬 저장 데이터는 Android 기기 보안(암호화)에 의해 보호됩니다.
  • 약품 복용 정보는 외부 서버로 전송되지 않으며, 오직 이용자 기기 내에만 저장됩니다.
  • AI 스캔을 위해 전송되는 이미지는 Google Gemini API 처리 후 즉시 삭제됩니다.

7개인정보의 파기 절차 및 방법

  • 파기 절차: 이용자가 앱 내에서 직접 삭제하거나, 앱 삭제 시 자동 파기됩니다.
  • 파기 방법: 전자적 파일 형태의 정보는 복구 불가능한 방법으로 영구 삭제합니다.

8행태정보 수집 및 광고 거부 방법

앱은 AdMob을 통해 맞춤형 광고를 제공합니다. 다음 방법으로 맞춤형 광고를 거부할 수 있습니다.

  • Android 설정 → 개인정보 보호 → 광고 → 광고 ID 재설정 또는 삭제
  • 앱 내 설정 화면에서 맞춤형 광고 동의 철회
거부 시 광고 자체가 사라지지 않으며, 비개인화 광고로 전환됩니다.

9개인정보의 국외 이전

앱은 AI 스캔 기능 사용 시 약 봉지 이미지를 Google LLC(미국)의 Gemini API 서버로 전송합니다.

이전받는 자이전 국가이전 항목이전 목적보유 기간
